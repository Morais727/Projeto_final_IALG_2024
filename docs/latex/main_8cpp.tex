\doxysubsection{Detailed Description}
Implementação de funções para leitura, processamento e ordenação de dados de acomodações. 

Este programa permite carregar, visualizar, modificar, excluir e exportar registros de acomodações armazenados em arquivos CSV e binários. Utiliza menus interativos e funcionalidades de {\ttfamily ncurses} para aprimorar a interface do usuário.
