Exibe os registros de acomodações em uma interface interativa utilizando ncurses. 

Esta função exibe uma tabela formatada com os detalhes das acomodações, permitindo ao usuário navegar pelos registros, alternar páginas e selecionar um item específico. A interface é construída utilizando a biblioteca ncurses para proporcionar uma experiência interativa e sofisticada.

\hline
{\em tamanho\+Inicial} & Índice do primeiro registro a ser exibido. \\
\hline
{\em tamanho\+Final} & Índice do último registro a ser exibido.\\
\hline
\end{DoxyParams}

\item ID do item selecionado (índice + 1).

\item -\/20 para indicar que o usuário escolheu filtrar.
\end{DoxyItemize}

\begin{DoxyItemize}
\item A função utiliza ncurses para criar uma interface interativa.
\item Os registros são exibidos em uma tabela formatada com colunas para ID, nome, quartos, banheiros, mínimo de noites, avaliações e preço.
\item O usuário pode navegar pelos registros usando as setas do teclado.
\item É possível alternar entre páginas de resultados usando as teclas \textquotesingle{}B\textquotesingle{} (anterior) e \textquotesingle{}N\textquotesingle{} (próxima).
\item A tecla \textquotesingle{}V\textquotesingle{} permite ao usuário voltar ao menu anterior.
\item A tecla \textquotesingle{}F\textquotesingle{} permite ao usuário acessar o menu de filtros.
\item A tecla \textquotesingle{}Enter\textquotesingle{} seleciona o item destacado.
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item A função inicializa e finaliza o modo ncurses automaticamente.
\item O cursor é ocultado durante a execução da função.
\item A função retorna o índice do item selecionado ajustado para começar em 1 (em vez de 0). 
\end{DoxyItemize}
\end{DoxyNote}
