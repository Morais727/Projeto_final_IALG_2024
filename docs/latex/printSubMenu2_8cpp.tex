\doxysubsection{Detailed Description}
Exibe o Sub Menu 2 e permite ao usuário interagir com as opções disponíveis. 

Esta função exibe um menu interativo com um cabeçalho estilizado e três opções. O cursor é ocultado durante a exibição do menu. O usuário pode navegar pelas opções e selecionar uma delas. A escolha do usuário é retornada como um valor inteiro.
