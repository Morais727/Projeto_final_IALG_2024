\doxysubsection{Detailed Description}
Definições de estruturas e declarações de funções auxiliares para manipulação de acomodações. 

Contém a estrutura {\ttfamily acomodacoes} e as assinaturas de funções utilizadas para operações como leitura, escrita, ordenação e exibição de dados.

Adiciona um novo registro de acomodação ao array e atualiza o arquivo binário.
