Carrega os registros de acomodações a partir de um arquivo binário. 

Se o arquivo binário não existir, a função inicializa um novo array dinâmico. Se o arquivo estiver corrompido, os dados não serão carregados e um novo espaço será alocado. A função utiliza alocação dinâmica para armazenar os registros.
