Detailed Description

Exibe uma barra de carregamento animada na interface ncurses.

Author
Marcos Morais
Date
03/02/2025

Esta função calcula dinamicamente a largura da barra de progresso com base no tamanho da tela e no progresso atual. A barra é centralizada e exibida no meio da tela.
