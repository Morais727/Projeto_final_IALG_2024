Altera um registro existente no array e atualiza o arquivo binário.

O usuário informa o ID do registro que deseja alterar. Se encontrado, os dados são exibidos e podem ser modificados. As alterações são salvas no arquivo binário.

Parameters
registrosPonteiro para o array de registros de acomodações.
tamanhoAtualNúmero atual de registros armazenados.
nomeArquivoNome do arquivo binário onde os registros são armazenados.
Returns
int Retorna 0 se a alteração for bem-sucedida ou se o usuário cancelar a operação, e -1 se ocorrer um erro na escrita do arquivo.
